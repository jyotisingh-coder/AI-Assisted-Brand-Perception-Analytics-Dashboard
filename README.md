# AI-Assisted Brand Perception Analytics Dashboard

## Project Overview
This project analyses the gap between intended brand identity and actual customer perception using AI-assisted sentiment analysis and hospitality review analytics.

The prototype focuses on identifying service areas where customer experiences do not align with the brand’s intended positioning.

## Objectives
- Analyse customer sentiment from hotel reviews
- Identify brand perception gaps
- Categorise hospitality service issues
- Generate strategic business recommendations
- Visualise alignment between brand identity and customer experience

## Tools & Technologies
- Python
- Google Colab
- Pandas
- TextBlob
- Matplotlib
- Seaborn
- AI-assisted sentiment analysis

## Workflow
1. Customer review collection
2. Sentiment analysis
3. Service category classification
4. Brand perception alignment analysis
5. Strategic recommendation generation
6. Data visualization

## Key Insights
- Identified misalignment areas affecting customer perception
- Analysed operational pain points influencing brand experience
- Generated AI-assisted strategic recommendations for service improvement
